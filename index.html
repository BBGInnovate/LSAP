Frequenty Asked Questions

To create a new media player you will need the following

  1. an IceCast HTTP streaming media server to stream the audio
  2. an audio source, either a live stream or a playlist of audio files
  3. encoding software with support for MP3 or AAC output
  4. a Web Server running PHP5 to host the media player application

How do I install a new media player on my website?

  1. Download the Zip file from the Developer Center by following the easy 'How To' steps in our guide to creating a new player
  2. Unzip the downloaded file and upload the entire media player folder to your Web Server. The folder will be the name of the new media player channel you created.
  3. Point your browser to http://yourwebsite.com/yourchannel.) Start Streaming on the Web and from your mobile device!

Can I make changes to my media player after I've created it?

Yes, all the files in your new channel folder are editable if you are very familiar with editing HTML, CSS, Javascript and PHP. However we suggest you use the Developer Center to easily create a new player, it's free to use and you can always create a new player.

Why do you need me to add my personal contact information to create a new player?

We'll only ever use this information to be able to contact you about future software updates and services that may become available to use with your player. We never share this information with 3rd party services or use it to track activity on the Web.

I'm a developer, how can I contribute to this open source project?

Follow the project's progress and make contributions on our GitHub Page.
